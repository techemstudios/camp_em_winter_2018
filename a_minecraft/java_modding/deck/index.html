Setting Up Forge in Eclipse

Before Creating Mods

Type the series of commands below in the command prompt or terminal:

$ cd Desktop

$ mkdir forge

$ cd forge

Move (or extract) the downloaded forge file into this new directory

Then, change to the directory using the "cd forge" command

Set Up a Forge Workspace

Command to Set Up Forge

on Windows: $ gradlew setupDecompWorkspace eclipse

on Mac OSX: $ ./gradlew setupDecompWorkspace eclipse

Setup a Directory as the Workpace

Install Eclipse and launch it!

Under the Worskpace Launcher

  • Click Browse
    • navigate to the forge directory
    • Select the eclipse folder inside
    • Click Open
    • Click OK
